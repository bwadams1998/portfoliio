Test Case 11: YouTube Video Search

Test ID: YT-TC-001
Priority: High
Module: Search

Preconditions:

  • User is on YouTube homepage
  • Internet connection active

Test Steps:

  1. Navigate to youtube.com
  2. Click on search bar at top
  3. Enter search query "guitar lessons"
  4. Press Enter or click search icon

Expected Result:

  • Search results page loads
  • Videos related to query displayed
  • Each result shows thumbnail, title, channel name, views, upload date
  • Filters available (Upload date, Type, Duration, Features, Sort by)
  • Search query appears in search bar
  • Related searches suggested

Test Data:

  • Search Query: "guitar lessons"

Test Case 12: Play Video

Test ID: YT-TC-002
Priority: High
Module: Video Player

Preconditions:

  • User is on YouTube
  • Search results or homepage loaded

Test Steps:

  1. Click on any video thumbnail
  2. Observe video behavior
  3. Verify video controls appear

Expected Result:

  • Video page loads
  • Video begins playing automatically (or requires user click based on settings)
  • Video player controls visible (play/pause, volume, settings, fullscreen)
  • Video title, channel name, view count, upload date displayed below video
  • Recommended videos appear on right sidebar
  • Video loads without buffering issues (with good internet)

Test Data:

  • Video: Any public video

Test Case 13: Subscribe to Channel

Test ID: YT-TC-003
Priority: Medium
Module: User Interaction

Preconditions:

  • User is logged into YouTube account
  • User is viewing a channel they are not subscribed to

Test Steps:

  1. Navigate to any YouTube channel
  2. Locate "Subscribe" button (usually red, below channel banner)
  3. Click "Subscribe" button
  4. Observe button change

Expected Result:

  • "Subscribe" button changes to "Subscribed"
  • Button color changes (red to gray)
  • Bell icon appears next to button for notifications
  • Subscription count increases by 1
  • Channel appears in user's "Subscriptions" feed
  • Confirmation animation may appear

Test Data:

  • Channel: Any public YouTube channel

Test Case 14: Like/Dislike Video

Test ID: YT-TC-004
Priority: Medium
Module: User Interaction

Preconditions:

  • User is logged in
  • Video is playing or loaded

Test Steps:

  1. Navigate to any video
  2. Locate like/dislike buttons below video
  3. Click "Like" button (thumbs up icon)
  4. Observe button state

Expected Result:

  • Like button becomes highlighted/filled
  • Like count increases by 1
  • User can toggle off by clicking again
  • If dislike previously clicked, it un-highlights
  • Like is saved to user's "Liked videos" playlist
  • Button state persists on page refresh

Test Data:

  • Video: Any public video

Test Case 15: Adjust Video Quality Settings

Test ID: YT-TC-005
Priority: Medium
Module: Video Player

Preconditions:

  • User is watching a video
  • Video player is visible

Test Steps:

  1. Start playing any video
  2. Click settings icon (gear icon) in video player controls
  3. Select "Quality"
  4. Choose different quality option (e.g., 1080p, 720p, 480p)
  5. Observe video quality change

Expected Result:

  • Settings menu opens
  • Quality options displayed (Auto, 2160p, 1440p, 1080p, 720p, 480p, 360p, etc.)
  • Selected quality is marked/highlighted
  • Video adjusts to selected quality within 1-2 seconds
  • Video may buffer briefly during quality change
  • New quality setting persists

Test Data:

  • Quality Change: From Auto to 1080p
