Test Case 16: Discord Login - Valid Credentials

Test ID: DISC-TC-001
Priority: High
Module: Authentication

What you need first:

  • A valid Discord account
  • Be on the Discord login page

Steps to test:

  1. Go to discord.com/login
  2. Enter your email or phone number
  3. Type in your password
  4. Click "Log In"

What should happen:

  • You get logged in successfully
  • Your Discord home screen loads with your server list
  • Your avatar and username show up
  • The server list appears on the left
  • Either your friends list or your last viewed channel opens
  • Your login session stays active

Test Data:

  • Email: testuser@example.com
  • Password: DiscordPass123!

Test Case 17: Send Direct Message

Test ID: DISC-TC-002
Priority: High
Module: Messaging

What you need first:

  • Be logged in
  • Have at least one friend or contact

Steps to test:

  1. Click on the "Friends" icon in the left sidebar
  2. Pick a friend from your list
  3. Type a message in the text field at the bottom
  4. Press Enter or click the Send icon

What should happen:

  • Your message appears in the chat right away
  • It shows your avatar and username
  • There's a timestamp
  • The message gets delivered to your friend
  • They might see a "Typing..." indicator while you were typing
  • The message stays in your chat history

Test Data:

  • Message: "Hello, this is a test message"

Test Case 18: Join Server via Invite Link

Test ID: DISC-TC-003
Priority: Medium
Module: Server Management

What you need first:

  • Be logged in
  • Have a valid invite link

Steps to test:

  1. Click the "+" button in your server list (left sidebar)
  2. Select "Join a Server"
  3. Paste the invite link in the text field
  4. Click "Join Server"

What should happen:

  • Discord checks that the invite is valid
  • You join the server successfully
  • The server shows up in your server list
  • You get taken to the server's default channel
  • You might see a welcome message (if they have one set up)
  • Any applicable server roles get assigned to you

Test Data:

  • Invite Link: discord.gg/example123

Test Case 19: Create Voice Channel

Test ID: DISC-TC-004
Priority: Medium
Module: Voice/Channel Management

What you need first:

  • Be logged in
  • Have admin or manage channel permissions on a server
  • Be viewing that server

Steps to test:

  1. Go to a server where you have the right permissions
  2. Right-click on the server name or a channel category
  3. Select "Create Channel"
  4. Choose "Voice Channel" as the type
  5. Type in a name for the channel
  6. Click "Create Channel"

What should happen:

  • The create channel dialog opens up
  • You can select voice channel as an option
  • You can type in whatever name you want
  • The new voice channel appears in the server's channel list
  • Users can access it based on their permissions
  • It shows a voice channel icon (speaker symbol)

Test Data:

  • Channel Name: "Test Voice Chat"

Test Case 20: React to Message with Emoji

Test ID: DISC-TC-005
Priority: Low
Module: User Interaction

What you need first:

  • Be logged in
  • Be viewing a text channel with messages

Steps to test:

  1. Hover over any message in the channel
  2. Click on the "Add Reaction" icon (emoji face with a plus sign)
  3. Pick an emoji from the picker
  4. Watch the reaction appear

What should happen:

  • The emoji picker opens
  • You can search through the emoji list
  • The emoji you picked appears below the message
  • The reaction count shows "1"
  • If someone else clicks the same emoji, the count goes up
  • You can remove your reaction by clicking it again
  • Everyone in the channel can see the reactions
