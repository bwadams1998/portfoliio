Test Case 1: User Login - Valid Credentials

Test ID: AMZ-TC-001
Priority: High
Module: Authentication

What you need first:

  • A valid Amazon account
  • Start at the Amazon homepage

Steps to test:

  1. Go to www.amazon.com
  2. Hit the "Sign in" button in the top right
  3. Type in your email or phone number
  4. Click "Continue"
  5. Enter your password
  6. Click "Sign in"

What should happen:

  • You'll get logged in successfully
  • The page takes you back to the homepage (or wherever you were before)
  • Your name shows up in the top right corner with a "Hello, [Your Name]" greeting

Test Data:

  • Email: testuser@example.com
  • Password: ValidPass123!

Test Case 2: Product Search Functionality

Test ID: AMZ-TC-002
Priority: High
Module: Search

What you need first:

  • Be on the Amazon homepage

Steps to test:

  1. Find the search bar at the top
  2. Type in "wireless headphones"
  3. Either click the search icon or just press Enter

What should happen:

  • Results load pretty quickly (within a few seconds)
  • You'll see relevant headphones that match what you searched for
  • Filters pop up on the left side (things like Brand, Price, Reviews)
  • There's a count showing how many results were found (like "1-48 of over 5,000 results")
  • Each product shows an image, title, price, and star rating

Test Data:

  • Search Query: "wireless headphones"

Test Case 3: Add Item to Cart

Test ID: AMZ-TC-003
Priority: High
Module: Shopping Cart

What you need first:

  • Be logged into your account
  • Be looking at a product page

Steps to test:

  1. Go to any product page
  2. Make sure you can see the "Add to Cart" button
  3. Pick a quantity if that's an option
  4. Choose any required options like size or color
  5. Click "Add to Cart"
  6. Watch for the confirmation

What should happen:

  • You get an "Added to Cart" confirmation message
  • The cart icon at the top updates to show you have items
  • You'll see options to either checkout or keep shopping
  • When you check your cart, the item is actually there

Test Data:

  • Product: Any eligible product
  • Quantity: 1

Test Case 4: Apply Promo Code at Checkout

Test ID: AMZ-TC-004
Priority: Medium
Module: Checkout

What you need first:

  • Be logged in
  • Have at least one thing in your cart
  • Have a valid promo code ready

Steps to test:

  1. Click the cart icon
  2. Hit "Proceed to Checkout"
  3. Confirm your shipping address is correct
  4. Move to the payment page
  5. Look for the "Add promo code" or "Gift card/Promo code" field
  6. Type in your promo code
  7. Click "Apply"

What should happen:

  • The promo code gets validated
  • Your discount gets applied to the total
  • You can see the new, lower price
  • There's a message saying "Your coupon has been applied"
  • The order summary clearly shows the discount

Test Data:

  • Promo Code: TEST10 (example)

Test Case 5: Filter Search Results by Price

Test ID: AMZ-TC-005
Priority: Medium
Module: Search/Filters

What you need first:

  • You've already searched for something
  • You're on the search results page

Steps to test:

  1. Search for "laptop"
  2. Find the price filter on the left sidebar
  3. Select the "$500 to $1000" price range
  4. Watch the results update

What should happen:

  • The page refreshes (or updates smoothly)
  • All the laptops shown now fall within your selected price range
  • Your filter choice is highlighted or checked
  • The URL changes to reflect what you filtered
  • The product count updates to match

Test Data:

  • Search Term: "laptop"
  • Price Range: $500-$1000
